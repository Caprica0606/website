PHP DATA TYPES

Data type

Definition

Examples

String

Characters

$genre = 'fps';

$color = 'red';

Integer

Whole number

$gameLevel = 10;

$totalPoints = 98;

Float

Decimal

$probability = .8;

$avg = -10.7;

Boolean

True or false

$win = true;

$quit = false;

Array

Stores Multiple Values

$cards = array ($hearts,$spades,$diamonds,$clubs)

Object

A type of data that contains variables and functions to define it

class Fruit {

function fruit() {

$this->model = "apple";

}

}

Null

empty

$newGame = NULL;
